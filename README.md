## 这是什么

微博克隆器是一款微博备份工具，适合需要保存个人微博历史内容的用户使用。它不是通用爬虫框架，定位是个人数据备份、个人内容归档、微博内容导出和本地资料沉淀。

常见使用场景：

- 账号迁移、停用或清理前，先保存个人微博历史资料
- 自媒体运营者整理过往图文、评论和转发素材
- 长期使用微博后，把分散在平台里的内容做本地归档
- 希望给重要微博内容、图片、视频留一份本地副本
